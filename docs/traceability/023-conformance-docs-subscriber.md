# Traceability: Conformance Docs and PubSub Subscriber

Feature 023 keeps public documentation current, adds stale-claim guards, and
adds a scoped UADP/UDP subscriber decoder for the same Data Key Frame subset
emitted by the existing publisher. The feature remains profile-targeting and
does not make a CTT or profile-compliance claim.
