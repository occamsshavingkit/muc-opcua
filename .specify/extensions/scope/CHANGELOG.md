# Changelog

## 1.0.0 (2026-04-11)

- Initial release
- Add `/speckit.scope.estimate` command for effort estimation from spec artifacts
- Add `/speckit.scope.compare` command for side-by-side spec scope comparison
- Add `/speckit.scope.creep` command for scope creep detection via git history
- Add `/speckit.scope.budget` command for time budget generation per phase
- Optional `after_specify` hook for auto-generating effort estimates
- Complexity scoring based on requirements, tasks, decisions, and integrations
