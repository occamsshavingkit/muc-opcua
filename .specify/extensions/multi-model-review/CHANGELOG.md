# Changelog

## 0.1.2 - 2026-06-19

### Added

- Codex OSS/local provider guidance for Ollama, LM Studio, and other Codex CLI-backed review or spec-authoring flows.
- `codex exec --oss` command examples for local reviewer and spec-author handoffs.

### Changed

- Clarified that Codex `--oss`, `oss_provider`, and custom `model_provider` settings live in Codex CLI configuration, not in the multi-model spec axis grammar or Claude Code subagent frontmatter.
- Updated Spec Kit catalog submission notes to the current Extension Submission issue workflow instead of direct catalog pull requests.
- Updated public release metadata and install documentation for the `v0.1.2` Spec Kit extension archive.

## 0.1.1 - 2026-06-09

### Added

- Optional work-assist orchestration guidance for OMO, UltraWork, lazycodex, and Superpowers-aware review flows.
- Headroom-aware package compression notes and measured RTK/Headroom completion reporting.
- Automatic subagent model routing guidance for scout, heavy-planner, worker, and review-checker task slices.

### Changed

- Updated public release metadata and install documentation for the `v0.1.1` Spec Kit extension archive.

## 0.1.0 - 2026-05-04

### Added

- Initial public extension package for multi-model Spec Kit review workflows.
- Spec authoring, review package, review ingestion, and cross-review routing commands.
- Claude Code subagent routing guidance for scout, planner, worker, and review-checker roles.
